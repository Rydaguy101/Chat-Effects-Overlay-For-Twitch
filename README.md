### Linux Installation

#### Step 1: Download AppImage
1. Go to the [Releases](https://github.com/Rydaguy101/Chat-Effects-Overlay-For-Twitch/releases) page
2. Download the latest `.AppImage` file*

#### Step 2: Make Executable and Run
1. Open terminal and navigate to your download directory
2. Make the AppImage executable:
```bash

```

3. Run the application:
```bash

```

#### Alternative: Package Manager Installation

**For Fedora/RHEL (.rpm package):**

### Step 1: Initial Launch

## üõ†Ô∏è Troubleshooting

### Common Issues

#### "Windows protected your PC" Warning
This is normal for new applications. Click "More info" ‚Üí "Run anyway"


#### Can't Connect to Twitch Chat
1. Check your internet connection
2. Verify your channel name is correct (no # symbol)
3. Try clicking "Reconnect"


#### Effects Not Showing
1. Make sure overlay mode is enabled
2. Check that keywords are properly configured
3. Verify the overlay is positioned correctly

#### Antivirus Blocking Installation
1. Temporarily disable real-time protection
2. Add the installer to your antivirus exclusions
3. You can scan the file with VirusTotal if concerned

### Performance Issues

#### High CPU Usage
1. Lower the effect quality in settings
2. Reduce the number of simultaneous effects
3. Close other unnecessary applications

#### Effects Lagging
1. Check if hardware acceleration is enabled
2. Update your graphics drivers
3. Consider lowering the overlay resolution
